## 150g Vertical Spinner Combat Robot

I have designed a 150g Combat Robot, utilising a 2207 motor, spinning a Titanium Blade. I have the goal to compete with it in February.

It uses a unibody Polypropylene chassis, allowing weight to be shifted to other areas (due to the low material density), while retaining a vast amount of impact resistance, allowing for it to be competitive in combat.
